## English

### Overview

RRdis is a high-performance desktop client for Redis management, designed for developers and architects who demand precision, speed, and a beautiful interface that stays out of your way.

### Features

- **Multi-tab Connection** - Switch between local, staging, and production clusters in milliseconds without losing context
- **Full Data Support** - Strings, Lists, Sets, Hashes, and Streams handled with native UI editors
- **Blazing Fast** - Native Tauri 2 core means zero overhead and instant startup times
- **Built-in CLI** - Full command autocomplete and history support for power users
- **Multi-language** - Localized for global teams, supporting 12+ languages out of the box
- **Modern UI** - A design system that respects your focus and reduces ocular fatigue

### Tech Stack

- **React 19** - Modern UI framework
- **TypeScript** - Type-safe development
- **Rust** - High-performance backend
- **Tauri 2** - Native desktop application framework

## 中文

### 项目简介

RRdis 是一款高性能的 Redis 桌面管理客户端，专为追求精准、速度和优雅界面的开发者和架构师设计。

### 功能特性

- **多标签连接** - 在本地、测试和生产环境集群之间毫秒级切换，不丢失上下文
- **完整数据支持** - 原生 UI 编辑器支持字符串、列表、集合、哈希和流
- **极速性能** - 基于 Tauri 2 原生内核，零开销，瞬间启动
- **内置命令行** - 为高级用户提供完整的命令自动补全和历史记录支持
- **多语言支持** - 为全球团队本地化，开箱即支持 12+ 种语言
- **现代化界面** - 尊重专注力的设计系统，减少视觉疲劳

### 技术栈

- **React 19** - 现代化 UI 框架
- **TypeScript** - 类型安全开发
- **Rust** - 高性能后端
- **Tauri 2** - 原生桌面应用框架
